# Release Roadmap

In ottica Agile, la pianificazione dettagliata avviene sprint by sprint all'interno di Jira. Tuttavia, a livello strategico, il progetto si articola su un orizzonte temporale di **8 mesi (16 Sprint)** suddiviso in 3 macro-release per garantire rilasci incrementali di valore e mitigare i rischi tecnici in anticipo.
